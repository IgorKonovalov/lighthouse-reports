lighthouse report generator

Add set of pages to index.js

```
const pageUrls = [
  {
    name: 'Google',
    url: 'https://google.com/',
  },
  {
    name: 'yandex',
    url: 'https://yandex.com/',
  },
  ...
];
```

Install dependencies and run

```
npm i
npm start
```

Look at the reports in "reports" folder
